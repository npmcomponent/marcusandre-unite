
# unite

  Easily merge arrays.

## Installation

  Install with [component(1)](http://component.io):

    $ component install marcusandre/unite

## API

  Merge some arrays into on array:

  ```js
  var unite = require('unite');
  var arr = unite(['2', '4', '8'], ['b', 'd', 'h']);
  // ['2', '4', '8', 'b', 'd', 'h']
  ```
